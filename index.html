核心功能介紹

針對現代考勤與活動打卡需求，打造最齊全的智慧防偽與高效架構

強制實名驗證

自動提取 Google 帳號真實姓名作為打卡依據，輸入框全面鎖定禁止手動更改，徹底杜絕假名與惡意代打。

智慧座標超連結

收集到的高精度經緯度不僅是數據，更是直接綁定 Google 地圖的超連結。在後台點擊座標，即可瞬間開啟地圖定位裝置確切位置。

120 秒防刷冷卻

內建 120 秒打卡冷卻時間，有效防止同一裝置惡意重複點擊。當後台資料庫一鍵清空時，冷卻狀態將立即同步重置，隨時準備迎接下一波活動。

雙語系與深淺模式

無縫支援繁體中文與英文即時切換，並內建深色 (Dark) 與淺色 (Light) 主題模式，完美適應不同國籍的用戶與活動現場的視覺偏好。

專屬網址，隨點即用

完全無需下載安裝任何 App。主辦方只需透過社群或群組分享專屬打卡網址，參加者點擊連結即可在手機瀏覽器內極速完成定位打卡，打造零摩擦的完美體驗。

一鍵無縫刷新數據

後台配備專屬的資料刷新按鈕，點擊即可瞬間動態載入最新打卡進度。無須重新載入整個網頁，確保操作極致流暢的同時，完美優化 API 請求次數以節省主機資源。

安全授權與 IP 追蹤

在符合瀏覽器安全規範下請求 GPS 權限，並同時在背景無感收集裝置 IP 位址，提供雙重數據交叉比對，讓假定位無所遁形。

100% Serverless 架構
